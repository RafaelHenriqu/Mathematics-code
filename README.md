## 📚 Sobre o projeto

O Mathematics Code foi uma biblioteca criada em **Node.js** com o objetivo de facilitar operações matemáticas e utilitárias dentro da programação.

A ideia na época era reunir diversas funções matemáticas e matemáticas aplicadas em um único pacote, tornando mais simples o uso em projetos.

---

## 🧠 Contexto de aprendizado

Este projeto foi desenvolvido em uma fase inicial da minha jornada como programador.

Na época:

* Eu ainda não tinha conhecimento de que o **JavaScript já possui diversas funções nativas que substituem grande parte dessas implementações**
* Muitas funcionalidades foram recriadas do zero sem necessidade prática real
* Em retrospecto, o projeto acabou sendo um exercício de aprendizado e experimentação

Um exemplo disso é a conversão de valores (como binário), que foi implementada manualmente, mesmo existindo formas muito mais simples na própria linguagem.

---

## 🧩 Uso de TypeScript (implementação inicial incorreta)

Este também foi um dos meus primeiros projetos onde tentei utilizar **TypeScript**.

No entanto, isso foi feito de forma totalmente experimental e sem base técnica adequada na época:

* Sem estudo prévio de TypeScript
* Sem planejamento de arquitetura
* Sem entendimento correto de tipagem e configuração
* Implementação direta “tentando fazer funcionar”

Como resultado, existem vários problemas relacionados à integração com TypeScript no projeto.

Hoje, entendo que essa abordagem não segue boas práticas, mas ela fez parte do meu processo de aprendizado.

---

## ⚙️ Funcionalidades (visão geral da época)

Entre as funções implementadas ou planejadas estavam:

* Número Pi
* Fatorial
* Sequência de Fibonacci
* Operações básicas (soma, subtração, multiplicação, divisão)
* Cálculo de porcentagem
* Conversão para binário
* Geração de números aleatórios
* Cálculo de IMC
* Funções estatísticas simples (média, etc.)
* Raiz quadrada
* Cálculo de área
* Outras funções matemáticas variadas
